# TaskFlow - Gerenciamento de Tarefas

## Descrição
Projeto em Python para gerenciamento de tarefas pelo terminal.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas
- Concluir tarefas
- Remover tarefas

## Estrutura

```
TaskFlow-Gerenciamento-Tarefas/
│
├── src/
│   ├── app.py
│   ├── manager.py
│   └── task.py
│
├── tests/
│   └── test_manager.py
│
├── requirements.txt
├── .gitignore
└── README.md
```

## Como executar

```bash
python src/app.py
```

## Testes

```bash
pytest
